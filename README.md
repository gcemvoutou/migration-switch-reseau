# Migration switch réseau — Centre culturel LVS (Mairie de Saint-Égrève)

Projet mené en autonomie dans le cadre de mon alternance BTS SIO (option SISR) : remplacement du switch réseau du Centre culturel LVS (cinéma municipal) et harmonisation du plan de VLANs avec le reste du parc réseau de la mairie.

**En bref :**
- Ancien équipement : switch Alcatel-Lucent, 52 ports (26×2), câblage désorganisé
- Nouveau matériel : switch Aruba 2530 (`LVS_197`)
- Contraintes fortes : aucune intervention pendant les horaires d'ouverture au public
- Projet mené d'avril à début septembre 2026, en parallèle du réaménagement de la banque d'accueil
